📐 How to Test

Method 1 - Browser DevTools:

  1. Press F12 to open DevTools
  2. Click the device icon (📱) to toggle device mode
  3. Select different device sizes (iPhone, iPad, etc.)
  4. Test each page and interaction

Method 2 - Resize Browser:

  1. Make your browser window very narrow (less than 768px)
  2. Check if layout adapts properly
  3. Test like buttons, navigation, etc.

🎯 Key Breakpoints

Tablet (768px and below): Main responsive layout changes

Mobile (480px and below): Extra compact design optimizations
